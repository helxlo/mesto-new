# Проект: Mesto 

------
Данный проект является каталогом интересных уголков нашей страны и первым шагом на пути к освоению js.  
------
В процессе создания данного проекта были реализованы такие технологии, как:  
- Адаптивная вёрстка
- Грид
- Флексбокс
- Позиционирование
- Появление поп-апов
- Валидация инпутов
- Применение методов ООП
------
